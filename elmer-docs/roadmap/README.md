# Roadmap

Product roadmap and planning documents.

## Structure

```
roadmap/
├── README.md           # This file
├── roadmap.json        # Structured roadmap data
├── current.md          # Current quarter focus
└── future.md           # Future quarters/themes
```

## Roadmap Format

```markdown
# Q[N] [Year] Roadmap

## Theme: [Quarter Theme]

### P0 - Must Do
| Initiative | Outcome | Status |
|------------|---------|--------|
| [name] | [what changes] | [status] |

### P1 - Should Do
| Initiative | Outcome | Status |
|------------|---------|--------|

### P2 - Could Do
| Initiative | Outcome | Status |
|------------|---------|--------|
```

## Commands

- `/roadmap` - View current roadmap
- `/status` - See active initiatives
