# AGENTS.md Documentation

Product-focused documentation for AI agents interacting with codebases.

## Purpose

AGENTS.md files help AI agents understand:
- The WHY behind code (problems solved, users served)
- User empathy (personas, fears, success criteria)
- Flow understanding (triggers, steps, outcomes)
- Trust factors (what builds/breaks user trust)

## Structure

Mirror the source structure:

```
agents-docs/
└── [repo-name]/
    └── [path]/
        └── AGENTS.md
```

## Commands

- `/agents` - Generate AGENTS.md documentation for a codebase area
