# Scenarios

User scenarios and use cases for testing and validation.

## Purpose

Scenarios describe specific situations users encounter, used for:
- Prototype validation
- Edge case identification
- Jury system evaluation

## Scenario Format

```json
{
  "id": "scenario-001",
  "name": "Scenario Name",
  "persona": "persona-name",
  "context": "What situation the user is in",
  "goal": "What they're trying to accomplish",
  "steps": [
    "Step 1",
    "Step 2"
  ],
  "success_criteria": "How we know it worked",
  "edge_cases": ["Edge case 1"]
}
```
