# Hypotheses

Track product hypotheses and assumptions here.

## Hypothesis Format

```markdown
# Hypothesis: [ID]

## Statement
We believe that [persona] will [behavior] if we [feature] because [insight].

## Status
[draft | committed | validated | invalidated]

## Evidence Required
- [What would prove this true?]
- [What would prove this false?]

## Current Evidence
- [Links to research, signals, or data]

## Related Initiative
[Link to initiative]
```

## Workflow

1. Create hypothesis in `draft/` folder
2. Move to `committed/` when ready to test
3. Move to `validated/` or `invalidated/` based on evidence
