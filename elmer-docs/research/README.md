# Research

Store general user research, competitive analysis, and discovery findings here.

## Structure

```
research/
├── interviews/           # User interview notes
├── competitive/          # Competitive analysis
├── surveys/              # Survey results
└── discovery/            # General discovery work
```

## Initiative-Specific Research

Research for specific initiatives should go in:
`initiatives/[initiative-name]/research.md`

This folder is for cross-cutting research that doesn't belong to a specific initiative.
