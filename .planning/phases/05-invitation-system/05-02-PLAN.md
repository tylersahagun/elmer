## Objective

Create the invitation UI including a Share button and invite modal where admins can enter email and select role.

- Invitation API endpoints working
- Workspace settings page exists
- Admin role check in place

**This plan adds:**
- Invite modal component
- Share/Invite button in settings
- Copy invite link functionality
- Pending invitations list
