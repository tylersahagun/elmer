# Visual Digest

Create a visual summary of current initiatives, prototypes, and progress.

## When to Use

- `/visual-digest` or `/publish-digest`
- Weekly stakeholder updates

## Workflow

- Compile key artifacts, prototypes, and metrics.
- Summarize in a visual-friendly markdown layout.
- Write to `pm-workspace-docs/status/` using `write_repo_files`.

## Output

- `pm-workspace-docs/status/visual-digest.md`
