# Design Companion

Offer UI feedback and design recommendations aligned to the current visual system.

## When to Use

- `/design` or `/design-system`
- UX critique requests

## Workflow

- Review current UI context and identify consistency gaps.
- Provide concrete adjustments for spacing, hierarchy, and interaction feedback.
- Summarize recommendations in a report and write to repo.

## Output

- `pm-workspace-docs/status/design-companion.md`
