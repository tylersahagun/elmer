# Weekly Briefs

Project-level weekly executive briefs generated by `/weekly-brief [project-name]`.

## Format

Each file is named `[project-name]-YYYY-WXX.md` (e.g., `chief-of-staff-2026-W09.md`).

## Sections

- **Done** — Completed this week (Linear closed + Slack shipped items)
- **Doing** — Active work in progress (Linear in-progress + meeting context)
- **Blockers** — Hard blockers needing resolution
- **Up Next** — What's planned for next week by owner

## Notion Mirror

Every brief is also published to the Notion Weekly Updates DB:
`https://www.notion.so/ask-elephant/310f79b2c8ac81699d3ff2bcebdc3d15`

Title format: `[Project Name] — Week of [Mon DD]`

## To Generate

```
/weekly-brief [project-name]    # One project
/weekly-brief --all              # All active projects
```
