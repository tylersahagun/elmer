# Activity Reports

This directory contains time-bounded activity reports generated by `/eod` (end-of-day) and `/eow` (end-of-week) commands.

## File Structure

```
activity/
├── activity-history.json     # Trend tracking data
├── digest-manifest.json      # Digest index/metadata
├── slack-block-kit-digests.md
├── digest/                   # Daily + weekly digests
├── eod/                      # End-of-day reports
├── eow/                      # End-of-week reports
├── product-updates/          # Product + engineering updates
├── rob/                      # Revenue-facing updates
└── visuals/                  # Visual exports for digests
```

## Commands

| Command | Description                          |
| ------- | ------------------------------------ |
| `/eod`  | Generate end-of-day activity report  |
| `/eow`  | Generate end-of-week activity report |

## What's Tracked

- **GitHub elephant-ai**: PRs merged/opened, commits
- **GitHub pm-workspace**: Doc updates, prototype iterations
- **Linear**: Issue state changes, cycle progress
- **Local**: Branch activity, status changes

## Report Contents

Each report includes:

1. **Summary** - Activity counts by source
2. **By Initiative** - What got done, what needs to continue
3. **Stakeholder Involvement** - Who should do what next
4. **Focus Recommendations** - Priority actions

## History Tracking

`activity-history.json` stores daily and weekly snapshots for trend analysis:

- Week-over-week velocity comparison
- Phase transition tracking
- Shipping rate trends
