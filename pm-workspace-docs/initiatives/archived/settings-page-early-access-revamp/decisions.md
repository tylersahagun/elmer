# Decisions: Settings Page & Early Access Revamp

## Release Lifecycle

- Stage order: Alpha -> Invite-only Beta -> Open Beta -> GA
- GA removes feature from beta list (no GA with flag)
- Documentation required before any customer-facing beta

## Settings Scope

- Workspace privacy settings apply to managers/owners only
- Users can mark meetings public, not private

## Beta Features UI

- Alpha section visible only when at least one alpha flag is enabled
- "Learn more" link only when docs exist
- Invite-only Beta and Open Beta share a Beta badge (tentative)

## Open Decisions

- Alpha self-toggle vs view-only for customers
- Integrations placement (Settings vs dedicated area)
