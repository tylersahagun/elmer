## 3. Key Features (Relevant to Agent Command Center)

### Revenue Platform / Deal Inspection

- **Clari Inspect**: 360° pipeline visibility, real-time deal health dashboard
- **Deal Inspection Agent**: AI flags risk in real time, highlights execution gaps
- **Advanced Opportunity Scores**: Built on activity data, CRM, conversation signals
- **Smart Summaries**: Instant deal and account health understanding
- **Pulse analytics**: Real-time pipeline health monitoring
- **Trend Analysis Agent**: Early red flag detection across pipeline
- **75,000+ revenue team members** use Inspect daily
- Direct path from inspection to action (seconds, not static reports)

**Sources:** [Clari Inspect](https://www.clari.com/products/inspect/), [Clari Inspect Tour](https://www.clari.com/products/inspect/product-tour/)

### Clari Copilot (formerly Wingman) — Real-Time AI Assistance

- **Real-time transcription** during customer calls
- **Live battlecards**, insights, and relevant info during calls — not post-call
- **Automatic capture**: Intent, objections, contacts, next steps → CRM
- **Smart Topics**: Complex concepts aligned with deal cycles; pattern spotting across similar deals
- **Battlecards, playbooks, gametapes** for coaching and onboarding
- **Smart CRM Suggestions**: One-click updates from Copilot or Slack
- **Claimed outcomes**: 40% shorter cycles, 30% higher win rates, 90% less CRM time, 25% better forecasting
- Desktop app with real-time coaching and collaboration

**Sources:** [Clari Copilot](https://www.clari.com/products/copilot/), [Community Copilot Desktop](https://community.clari.com/product-q-a-6/boost-your-productivity-with-real-time-coaching-and-collaboration-introducing-copilot-formerly-wingman-s-new-version-of-our-desktop-app-569)

- Review **pipeline and priorities** in Clari dashboards
- **Groove Daily Digest Email**: Actions due or past due
- Sync with managers on key deals
- "Power hour" for cold calls, LinkedIn
- Prep client meetings with **Copilot** (past deal activity, stakeholders)

### Midday (11:00 AM–1:00 PM)

- Focus on high-impact deals
- **Omnibar** in inbox: Salesforce data, activity history, LinkedIn — no tool switching

### Throughout

- **Copilot** joins meetings; real-time transcription, battlecards, objections, next steps
- **Auto-capture** logs emails, meetings, contacts to CRM
- **Deal Inspection Agent** flags risk; **Trend Analysis Agent** surfaces patterns
- **Ask Clari** for instant answers and next-best-actions

## AskElephant Differentiation vs Clari

| Dimension            | Clari                                                                | AskElephant (Agent Command Center)                                       |
| -------------------- | -------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| **Entry point**      | Enterprise revenue platform; CRM/Salesforce-centric                  | Meeting-first; conversation as primary trigger                           |
| **Persona**          | CRO/RevOps/leader-first; rep adoption uneven                         | Rep-first viral anchor; action-over-insight                              |
| **Proactive hub**    | Groove digest + Omnibar; modules and configuration                   | Single command center; time-aware, dynamic UX                            |
| **Meeting clearing** | Copilot during/after call; separate from daily hub                   | Rapid-fire meeting clearing; "boom boom boom"                            |
| **AI configuration** | Revenue Cadences, agents, admin-driven                               | Chat-based config; "settings are not toggles... It's a chat... AI first" |
| **Trust / evidence** | Smart Summaries, Deal Inspection; less emphasis on source visibility | Evidence-backed outcomes; privacy-before-trigger                         |

**Strategic note:** Clari competes in the same "daily proactive hub + meeting intelligence + CRM automation" space but from an **enterprise platform** posture. Agent Command Center differentiates with **rep-first UX**, **meeting-clearing as anchor**, **action-over-insight**, and **chat-based AI config** — addressing rep friction that Clari reviews cite ("clunky," "negative ROI" for frontline).
