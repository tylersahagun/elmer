## Differentiation Summary

### Capabilities Where AskElephant Leads or Will Lead

| Capability | Category | Competitive Gap |
| --- | --- | --- |
| Property-first CRM config (no prompts) | **Unique** | No competitor offers structured UI for meeting-to-CRM field extraction |
| Preview before CRM sync | **Unique** | Zero competitors offer pre-write verification |
| Before/after field diff | **Unique** | Novel in CRM context (borrowed from code review UX) |
| Confidence thresholds | **Unique** | No competitor allows per-field confidence gating |
| Structured signal ontology | **Unique** (Phase 2) | Closest is Day.ai Context Graph, but from CRM data not meeting depth |
| Privacy determination | **Unique** | No competitor offers privacy-first capture model |
| HubSpot-native depth | **Opportunity Gap** | Most competitors are Salesforce-first; AskElephant owns HubSpot |
| CRM readiness audit | **Opportunity Gap** | No researched competitor offers CRM auditing |
