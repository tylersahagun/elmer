## Recommendations

1. **Prioritization review** — 21 projects In Progress may indicate capacity overload. Consider cycle planning to limit WIP.

4. **PM initiative mapping** — Cross-reference with `pm-workspace-docs/initiatives/active/*/_meta.json` for `linear_project_id` coverage.

## Methodology

1. Called `COMPOSIO_SEARCH_TOOLS` for Linear project listing
2. Executed GraphQL via `LINEAR_RUN_QUERY_OR_MUTATION`:
   ```graphql
   query { projects(first: 100) { nodes { id name state status { name } teams { nodes { id key name } } } } }
   ```
3. Filtered out projects where `status.name` in (Delivered, Released, Canceled)
4. Grouped by state and team
