# Global Chat: Internal Q and A Guide (Revenue Team)

## TL;DR

Global Chat is the AI chat experience that stays available across the app so users can ask questions about meetings, accounts, and their team. It works best when opened from the most relevant page.

## When to recommend Global Chat

- Reps prepping for a call or trying to recall prior conversations.
- Leaders coaching across multiple reps or accounts.
- CSMs preparing for renewals or escalations.
- RevOps auditing capture coverage or looking for patterns.

## Key points (internal)

- Always-available chat across the app for questions about meetings, accounts, and workflows.
- Works best when opened from the most relevant page (company, meeting, contact).
- If Internal Search is enabled, it can look across meeting history for deeper answers.

## Common questions and simple answers

**Where do I find it?**  
Web: Open Chat from the top navigation or use the keyboard shortcut (Cmd/Ctrl + K new, Cmd/Ctrl + L legacy).  
Mobile: Open Global Chat from the home screen prompt.

**Is this the same as Internal Search?**  
Global Chat is the interface. Internal Search is an optional tool that helps it search across meeting history.

**Why are answers better on some pages?**  
Global Chat uses the page context (company, meeting, contact) to be more specific. Open it from the most relevant page.

**Does it work on mobile?**  
Yes, when Mobile Global Chat is enabled for the workspace.

**Why does chat close when I navigate?**  
Some workspaces have an early access layout that keeps chat open across navigation. Others use the overlay that closes when you move to a new page.

**I do not see the Chat button.**  
Try the keyboard shortcut first. If chat still does not appear, the feature is likely not enabled for that workspace. Send the workspace name and user email to support for enablement.

**Answers feel too generic.**  
Ask from the most relevant page, add a time window, and make sure Internal Search is enabled.

**Which shortcut opens it?**  
New experience: Cmd/Ctrl + K. Some workspaces still use Cmd/Ctrl + L.

## Escalation checklist (keep it simple)

If a customer is blocked, collect:

- Workspace name
- User email
- Web or mobile
- What they expected vs what happened
- Screenshot if possible

## Internal enablement notes (for Product/Support)

- Feature flags: `global-chat-enabled`, `global-chat-exp-v2`, `global-chat-column-layout`, `mobile-global-chat-enabled`
- Early access feature: Global Chat Layout (persistent right-side column)
- Dashboards: Global Chat and Internal Search, Global Chat
