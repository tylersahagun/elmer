# PM Workspace Redaction Checklist (External Version)

Use this checklist whenever producing a public-facing guide or video.

---

## Remove or Replace

- Slack IDs, channel names, and permalinks
- MCP server names and tool identifiers
- Customer names, internal project codenames
- Internal file paths if they reveal private structure
- Team org chart details and reporting lines
- Internal metrics, revenue numbers, or pipeline details
- Anything marked as internal-only or gated by approval

---

## Replace With

- Generic tool references (“task tracker” instead of “Linear”)
- Redacted example names (“Customer A”)
- High-level workflow steps instead of command names

---

## Visual Hygiene

- Blur or crop personal data
- Remove identifiable Slack handles
- Avoid showing private URLs in screen recordings
